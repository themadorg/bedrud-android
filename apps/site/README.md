# Bedrud Site

Astro landing site and docs for [bedrud.org](https://bedrud.org).

## Development

```bash
bun install
bun dev          # dev server
bun run build    # production build (521 pages)
bun run check    # Biome lint + format
bun run typecheck:astro  # Astro type checking
```

See root [CONTRIBUTING.md](../../CONTRIBUTING.md) and [AGENTS.md](../../AGENTS.md) for monorepo conventions.
