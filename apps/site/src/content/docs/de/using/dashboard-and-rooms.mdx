Nach der Anmeldung ist das Dashboard die zentrale Stelle, um Räume zu erstellen, beizutreten und kürzlich genutzte Räume wiederzufinden.

## Raum erstellen

1. Erstellen Sie vom Dashboard aus einen neuen Raum (Name und ggf. Optionen, die Ihr Host freigibt).
2. Der Raum erscheint in Ihrer Liste.
3. Öffnen Sie den Raum, um in den Pre-Join- / Meeting-Flow zu gelangen.

Raumkarten unterstützen Beitreten und Löschen. Die Layoutdichte kann je nach UI-Version variieren; Beitreten und Löschen bleiben verfügbar.

## Schnellbeitritt

Fügen Sie einen **Raum-Slug** oder eine **vollständige Meeting-URL** in den Schnellbeitritt ein. Bedrud löst den Raum auf und führt Sie zum richtigen Beitrittspfad.

Teilbare Links sehen typischerweise so aus:

```
https://your-host/m/<room-id-or-slug>
```

## Letzte Räume

Nachdem Sie einem Raum **beitreten**, erscheint er unter **Letzte Räume**.

- Die Liste **übersteht einen vollständigen Seiten-Reload**.
- **Entfernen** Sie einen Eintrag — er bleibt nach dem Reload weg.
- Über eine Karte öffnen Sie denselben Raum schnell erneut.

Letzte Räume sind eine Client-Komfortliste; sie ersetzen nicht die serverseitige Raumliste, die Sie besitzen oder freigegeben bekommen.

## Startseite

Wenn Sie die Root-URL der Instanz öffnen:

- **Abgemeldet** — Marketing- / Anmeldepfad (kein defekter API-Proxy)
- **Angemeldet** — Einstieg ins Dashboard
