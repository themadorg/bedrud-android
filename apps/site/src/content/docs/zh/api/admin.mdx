管理员 endpoint 位于 `/api/admin/` 下。所有 endpoint 都需要 **superadmin** 访问级别，并在 `Authorization: Bearer <token>` header 中提供有效的 JWT。

## 概述

| Endpoint | 方法 | 描述 |
|---|---|---|
| `/api/admin/users` | GET | 列出所有用户（分页） |
| `/api/admin/users/:id` | GET | 获取用户详情及房间 |
| `/api/admin/users/:id/status` | PUT | 激活或停用用户 |
| `/api/admin/users/:id/accesses` | PUT | 更新用户的访问级别 |
| `/api/admin/users/:id` | DELETE | 永久删除用户 |
| `/api/admin/rooms` | GET | 列出所有房间（分页） |
| `/api/admin/rooms/:roomId` | PUT | 更新任意房间的设置 |
| `/api/admin/rooms/:roomId` | DELETE | 关闭房间（级联删除） |
| `/api/admin/rooms/:roomId/suspend` | POST | 暂停活跃房间 |
| `/api/admin/rooms/:roomId/participants` | GET | 从 LiveKit 获取实时参与者数据 |
| `/api/admin/rooms/:roomId/participants/:identity/kick` | POST | 将任意参与者踢出房间 |
| `/api/admin/rooms/:roomId/participants/:identity/mute` | POST | 将任意参与者的音频静音 |
| `/api/admin/settings` | GET | 获取系统设置（密钥已隐藏） |
| `/api/admin/settings` | PUT | 更新系统设置 |
| `/api/admin/invite-tokens` | GET | 列出所有邀请令牌 |
| `/api/admin/invite-tokens` | POST | 创建新的邀请令牌 |
| `/api/admin/invite-tokens/:id` | DELETE | 删除邀请令牌 |
| `/api/admin/online-count` | GET | 活跃参与者总数 |
| `/api/admin/livekit/stats` | GET | LiveKit 服务器聚合统计 |
| `/api/admin/cert-info` | GET | TLS 证书元数据 |

## 访问级别

Bedrud 使用多级访问系统。每个用户都有一个 `accesses` 数组，可以同时包含多个访问级别。

| 访问级别 | 描述 | 中间件保护 |
|---|---|---|
| `superadmin` | 完整系统访问。管理面板、用户/房间管理、设置、邀请令牌。 | `RequireAccess("superadmin")` 守卫 `/api/admin/*` |
| `admin` | 已定义但未由中间件强制执行。仅用于信息标识。 | 当前无 |
| `moderator` | 房间级管理。在单个房间内禁言聊天、关闭/开启音频、聚光灯、禁用视频。 | 房间范围内的 `isRoomModerator()` 检查 |
| `user` | 标准认证用户。创建/加入房间，管理个人资料。 | 用户路由上的 `Protected()` |
| `guest` | 用于加入公共房间的临时用户。仅限访客登录。 | 不适用 |

> **注意：** `/api/admin/*` 路由组检查 `superadmin` 而非 `admin`。只有 `accesses` 数组中包含 `superadmin` 的用户才能使用这些 endpoint。

## 用户管理

### 列出所有用户

获取所有已注册用户的分页列表。

```
GET /api/admin/users?page=1&limit=50
```

**Headers：** `Authorization: Bearer <accessToken>`

**查询参数：**

| 参数 | 类型 | 默认值 | 描述 |
|---|---|---|---|
| `page` | int | 1 | 页码 |
| `limit` | int | 50 | 每页用户数 |

**响应（200）：**

```json
{
  "users": [
    {
      "id": "123e4567-e89b-12d3-a456-426614174000",
      "email": "user@example.com",
      "name": "John Doe",
      "provider": "local",
      "isActive": true,
      "isAdmin": false,
      "accesses": ["user"],
      "createdAt": "2025-01-15 10:30:00"
    },
    {
      "id": "123e4567-e89b-12d3-a456-426614174001",
      "email": "admin@example.com",
      "name": "Admin User",
      "provider": "local",
      "isActive": true,
      "isAdmin": true,
      "accesses": ["user", "superadmin"],
      "createdAt": "2025-01-10 08:00:00"
    }
  ],
  "total": 42,
  "page": 1,
  "limit": 50
}
```

### 更新用户状态

激活或停用用户账户。被停用的用户无法登录或刷新令牌。

```
PUT /api/admin/users/:id/status
```

**路径参数：**

| 参数 | 描述 |
|---|---|
| `id` | 用户 UUID |

**请求体：**

```json
{
  "active": false
}
```

**响应（200）：**

```json
{
  "message": "User status updated successfully"
}
```

---

### 更新用户访问级别

更改用户的访问级别数组。用于提升为 superadmin、降级或添加 moderator 访问。

```
PUT /api/admin/users/:id/accesses
```

**路径参数：**

| 参数 | 描述 |
|---|---|
| `id` | 用户 UUID |

**请求体：**

```json
{
  "accesses": ["user", "superadmin"]
}
```

| 字段 | 类型 | 描述 |
|---|---|---|
| `accesses` | string[] | 访问级别数组。有效值：`user`、`superadmin`、`admin`、`moderator`、`guest` |

**响应（200）：**

```json
{
  "message": "User accesses updated"
}
```

---

### 删除用户

永久删除用户及其所有关联数据。异步执行 — 立即返回 `202 Accepted`。不能删除自己的账户。

```
DELETE /api/admin/users/:id
```

**路径参数：**

| 参数 | 描述 |
|---|---|
| `id` | 用户 UUID |

**响应（202）：**

```json
{
  "message": "User deletion queued",
  "rooms": 3
}
```

| 字段 | 描述 |
|---|---|
| `message` | 确认删除正在进行中 |
| `rooms` | 该用户创建的房间数量（也会被删除） |

后台任务级联删除以下内容：
- 用户创建的所有房间
- 关联的通行密钥（WebAuthn 凭据）
- 用户偏好设置
- 用户记录本身

## 房间管理

### 列出所有房间

获取实例上所有房间的分页列表（绕过创建者筛选）。

```
GET /api/admin/rooms?page=1&limit=50
```

**查询参数：**

| 参数 | 类型 | 默认值 | 描述 |
|---|---|---|---|
| `page` | int | 1 | 页码 |
| `limit` | int | 50 | 每页房间数 |

**响应（200）：**

```json
{
  "rooms": [
    {
      "id": "room-uuid-1",
      "name": "team-standup",
      "createdBy": "user-uuid",
      "maxParticipants": 25,
      "isActive": true,
      "settings": {
        "allowChat": true,
        "allowAudio": true,
        "allowVideo": true,
        "isPersistent": false
```

### 更新房间

修改任意房间的设置，包括仅管理员可用的 `isPersistent` 标志。

```
PUT /api/admin/rooms/:roomId
```

**路径参数：**

| 参数 | 描述 |
|---|---|
| `roomId` | 房间 UUID |

**请求体：**

```json
{
  "maxParticipants": 50,
  "settings": {
    "allowChat": true,
    "allowAudio": true,
    "allowVideo": true,
    "requireApproval": false,
    "e2ee": false,
    "isPersistent": true
  }
}
```

| 字段 | 类型 | 描述 |
|---|---|---|
| `maxParticipants` | int | 允许的最大参与者数（可选） |
| `settings.allowChat` | bool | 启用房间聊天 |
| `settings.allowAudio` | bool | 启用音频轨道 |
| `settings.allowVideo` | bool | 启用视频轨道 |
| `settings.requireApproval` | bool | 加入房间需要管理员批准 |
| `settings.e2ee` | bool | 启用端到端加密 |
| `settings.isPersistent` | bool | **仅管理员。** 阻止空闲房间被自动清理 |

所有设置字段均为可选 — 仅包含您想要更改的字段。

**响应（200）：** 返回更新后的房间对象。

---

### 关闭房间

永久关闭房间并进行级联删除。向所有参与者发送系统消息："This room has been closed by an administrator"。

```
DELETE /api/admin/rooms/:roomId
```

**路径参数：**

| 参数 | 描述 |
|---|---|
| `roomId` | 房间 UUID |

**响应（200）：**

```json
{
  "status": "success"
}
```

---

### 暂停房间

停用一个活跃房间及其所有参与者，但不删除房间记录。当有参与者重新加入时房间可以被重新激活。

```
POST /api/admin/rooms/:roomId/suspend
```

**路径参数：**

| 参数 | 描述 |
|---|---|
| `roomId` | 房间 UUID |

**响应（200）：**

```json
{
  "status": "success"
}
```

**错误（400）：** 如果房间已处于非活跃状态，返回 `{"error": "Room is not active"}`。

## 邀请令牌

邀请令牌用于实现门控注册。当设置中启用了 `tokenRegistrationOnly` 时，新用户在注册时必须提供有效且未使用的邀请令牌。

### 列出邀请令牌

获取所有邀请令牌，按最新优先排序。每个令牌包含一个由 `usedAt` 计算得出的 `used` 布尔值。

```
GET /api/admin/invite-tokens
```

**响应（200）：**

```json
{
  "tokens": [
    {
      "id": "token-uuid-1",
      "token": "a1b2c3d4e5f6...",
      "email": "invited@example.com",
      "createdBy": "admin-uuid",
      "expiresAt": "2025-06-15T10:30:00Z",
      "usedAt": null,
      "usedBy": "",
      "createdAt": "2025-03-15T10:30:00Z",
      "used": false
    },
    {
      "id": "token-uuid-2",
      "token": "f6e5d4c3b2a1...",
      "email": "",
      "createdBy": "admin-uuid",
      "expiresAt": "2025-04-15T10:30:00Z",
      "usedAt": "2025-03-20T14:00:00Z",
      "usedBy": "new-user-uuid",
      "createdAt": "2025-03-12T10:30:00Z",
      "used": true
    }
  ]
}
```

### 创建邀请令牌

生成一个新的邀请令牌。令牌值为 32 位随机十六进制字符串。

```
POST /api/admin/invite-tokens
```

**请求体：**

```json
{
  "email": "invited@example.com",
  "expiresInHours": 168
}
```

| 字段 | 类型 | 必填 | 默认值 | 描述 |
|---|---|---|---|---|
| `email` | string | 否 | _无_ | 将令牌限制为此邮箱地址 |
| `expiresInHours` | int | 否 | 72 | 令牌有效时长（小时） |

**响应（201）：**

```json
{
  "id": "uuid-string",
  "token": "a1b2c3d4e5f6a7b8c9d0e1f2a3b4c5d6",
  "email": "invited@example.com",
  "createdBy": "admin-uuid",
  "expiresAt": "2025-03-22T10:30:00Z",
  "usedAt": null,
  "usedBy": "",
  "createdAt": "2025-03-15T10:30:00Z"
}
```

| 参数 | 描述 |
|---|---|
| `id` | 邀请令牌 UUID |

**响应（200）：**

### 证书信息

获取 TLS 证书元数据，包括过期状态。

```
GET /api/admin/cert-info
```

**响应（200）— TLS 已启用，证书有效：**

```json
{
  "enabled": true,
  "status": "valid",
  "daysRemaining": 78,
  "notAfter": "2025-07-30T12:00:00Z",
  "subject": "CN=bedrud.example.com",
  "issuer": "CN=R3,O=Let's Encrypt,C=US",
  "notBefore": "2025-04-01T12:00:00Z",
  "sans": ["bedrud.example.com", "www.bedrud.example.com"]
}
```

**响应（200）— TLS 未启用：**

```json
{
  "enabled": false,
  "status": "not_configured"
}
```

| 状态 | 含义 |
|---|---|
| `valid` | 证书有效且未接近过期 |
| `expiring_soon` | 证书将在 30 天内过期 |
| `expired` | 证书已过期 |
| `not_configured` | TLS 已禁用 |
| `error` | 证书文件缺失或无效 |

## 错误响应

```json
{
  "error": "forbidden"
}
```

| 状态码 | 含义 |
|---|---|
| 400 | 无效输入（如错误的 JSON、缺少必填字段、删除自己的账户、暂停已非活跃的房间） |
| 401 | 未认证 — JWT 缺失或已过期 |
| 403 | 非 superadmin — 用户的 `accesses` 数组中缺少 `superadmin` |
| 404 | 资源未找到（用户、房间或令牌） |
| 409 | 冲突 — 用户不能删除自己的账户 |
| 500 | 内部服务器错误 |
| 501 | 未实现（如管理房间令牌生成） |
| 503 | 证书信息不可用（TLS 错误） |
