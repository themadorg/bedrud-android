登录后，仪表板是创建房间、加入房间以及返回最近使用房间的主入口。

## 创建房间

1. 从仪表板创建新房间（名称以及主机暴露的任何房间选项）。
2. 房间会出现在你的列表中。
3. 打开房间进入预加入 / 会议流程。

房间卡片支持加入与删除。布局密度可能随 UI 版本变化，但加入与删除始终可用。

## 快速加入

将**房间 slug** 或**完整会议 URL** 粘贴到快速加入。Bedrud 会解析房间并带你到正确的加入路径。

可分享链接通常类似：

```
https://your-host/m/<room-id-or-slug>
```

## 最近房间

**加入**房间后，它会添加到**最近房间**。

- 列表在整页**重新加载后仍保留**。
- **移除**某条最近记录——刷新后仍保持已移除。
- 使用最近卡片可快速重新打开该房间。

最近房间是客户端的便捷列表；它不会替代你拥有或可访问的服务端房间列表。

## 落地页

打开实例根路径时：

- **已退出** — 营销 / 登录路径（无损坏的 API 代理）
- **已登录** — 进入仪表板
