title: 舞台模式

舞台模式会切换房间布局，突出演示者（或舞台设置），而不是普通网格或聚光灯视图。

## 启动舞台

1. 作为主持人（或有权控制舞台的角色），从会议控制中**启动舞台**。
2. 其他参与者会看到**舞台布局**以及为舞台会话设计的**加入通知**。
3. 参与者条 / 侧栏仍可显示房间其余人员（若有开关则切换）。

## 舞台上的屏幕共享

- 舞台激活时共享屏幕——他人会看到**舞台屏幕共享叠加层**。
- **停止共享**会清除叠加层。

非舞台屏幕共享（普通网格）见 [会议控制](/zh/docs/using/meeting-controls)。

## 结束舞台

**结束舞台**将房间恢复为普通网格或聚光灯布局。

## 迟到加入者

在舞台**已激活时加入**的人应立即看到正确的舞台状态——而不是永远空白的网格。若迟到者布局错误，先刷新一次；若仍失败，请作为回归问题报告。

## 提示

- 舞台适用于演示、教学和一对多演示流程。
- 仅在叠加层设计为可共存时，再与 [YouTube 一起看](/zh/docs/using/youtube-watch) 或 [白板](/zh/docs/using/whiteboard) 组合；谨慎切换，避免参与者卡在多层全屏界面下。
