YouTube 一起看让一名参与者分享视频，房间内所有人一起观看。

## 启用功能

YouTube 观影派对可能位于 **设置 → 实验性 → YouTube 观影派对** 之后。在期望出现分享控制前先启用。

## 分享视频

1. 从会议控制打开**分享** / YouTube 对话框。
2. 粘贴**有效的 YouTube URL**。
3. 观看**叠加层**应在你和其他房间参与者上播放。

## 播放同步

播放、暂停与跳转（按实现）应对远端观众**大致保持同步**。网络较差时不保证帧级锁定；可能出现小漂移。

## 错误与关闭

| 情况 | 预期 |
|------|------|
| 无效 URL | 明确错误；不会空白挂起 |
| 关闭观看 | 双方叠加层消失 |
| 与舞台 / 白板切换 | 不应永久破坏这些模式 |
