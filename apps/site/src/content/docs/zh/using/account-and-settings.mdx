创建账户，保持会话，并在加入通话前配置个人资料与媒体偏好。

## 注册与登录

1. 在浏览器中打开你的 Bedrud 实例。
2. 使用邮箱和密码**注册**（以及主机要求的任何邀请令牌）。
3. 若启用了邮箱验证，在界面提示时完成验证步骤。
4. 使用有效凭据**登录**。密码错误会显示明确错误，且不会创建会话。
5. 从账户菜单**退出登录**。受保护路由（仪表板、设置、管理）会重定向到认证页。

只要访问令牌仍然有效，会话在整页**刷新**后仍会保留。

### 其他登录方式

取决于实例配置，你还可能看到：

- **OAuth**（Google、GitHub、Twitter 或自定义 OIDC）
- **通行密钥**（WebAuthn）——在安全设置中登记后可用
- **访客加入**——允许访客的房间无需账户（见 [加入会议](/zh/docs/using/joining-a-meeting)）

忘记密码？在登录页使用**忘记密码**，然后按重置链接操作（服务器需配置 SMTP）。

运维细节（JWT、通行密钥、速率限制）见 [身份认证](/zh/docs/backend/authentication)。

## 个人资料

从仪表板打开 **设置 → 个人资料**。

- 编辑**姓名** / 显示字段并保存——刷新后值会正确重新加载。
- 更改会出现在仪表板、会议画面和参与者列表中。

## 头像

仍在 **设置 → 个人资料** 下：

| 规则 | 说明 |
|------|------|
| 格式 | JPEG、PNG、WebP |
| 大小限制 | 小于 **2 MB** |
| 过大 / MIME 不正确 | 返回错误并拒绝（不会崩溃） |
| 重新上传 | 替换先前文件 |

上传后预览会立即更新。请在仪表板刷新后以及会议中确认新头像。

## 设置面板

从仪表板设置路由打开各面板：

| 面板 | 作用 |
|------|------|
| **个人资料** | 姓名与显示身份 |
| **音频** | 麦克风 / 扬声器列表；**一键通讲**启用 + 按住键捕获；跨刷新保存 |
| **视频** | 摄像头设备选择；跨刷新持久化 |
| **安全** | 修改密码与通行密钥登记入口 |
| **外观** | 主题及相关 UI 开关（若显示） |
| **实验性** | 可选功能，如**共享白板**与 **YouTube 一起看** |

偏好会保存在你的账户中（加入时可能同步到会议外壳）。通话内 PTT 行为见 [音频与一键通讲](/zh/docs/using/audio-and-push-to-talk)。
