配置麦克风与扬声器，然后可选使用一键通讲（PTT），仅在按住键时发送。

## 设备选择

在 **设置 → 音频**（以及预加入界面）：

- 浏览器授权后，麦克风与扬声器列表应填充。
- **通话中途**切换设备应无需完整重连房间。

视频摄像头在 **设置 → 视频**。见 [账户与设置](/zh/docs/using/account-and-settings)。

## 启用一键通讲

1. 打开 **设置 → 音频**。
2. 启用**一键通讲**。
3. 用按键捕获控件捕获**按住键**。
4. 若 UI 要求显式保存则保存——偏好应在刷新后持久化。

## 会议中

| 步骤 | 预期 |
|------|------|
| 按住 PTT 键 | 麦克风**发送**；他人听到你 |
| 松开键 | 麦克风**静音**；他人停止听到你 |
| 指示器 | 本地控制与远端画面在实现时可能显示 PTT 状态 |
| 禁用 PTT | 恢复普通静音切换行为 |

不要在希望保持安静时仍开启 PTT 导致麦克风卡住打开——若发生，禁用 PTT 或明确静音。

## 闭耳 / 耳机

若构建提供**闭耳**或耳机控制，不得让远端音频对他人卡住。闭耳是本地收听状态，不是远程静音他人（除非与管理操作结合）。
