YouTube Watch permet à un participant de partager une vidéo pour que tout le monde dans la salle regarde ensemble.

## Activer la fonction

La watch party YouTube peut être derrière **Paramètres → Expérimental → YouTube watch party**. Activez-la avant d'attendre l'apparition du contrôle de partage.

## Partager une vidéo

1. Ouvrez le dialogue **partage** / YouTube depuis les contrôles de réunion.
2. Collez une **URL YouTube valide**.
3. L'**overlay** de visionnage doit lire pour vous et pour les autres participants de la salle.

## Sync de lecture

Lecture, pause et seek (selon l'implémentation) doivent **rester à peu près synchronisés** pour les spectateurs distants. Un verrouillage image par image n'est pas garanti sur de mauvais réseaux ; un léger décalage peut apparaître.

## Erreurs et fermeture

| Situation | Attendu |
|-----------|---------|
| URL invalide | Erreur claire ; pas de blocage vide |
| Fermer le visionnage | Overlay disparu des deux côtés |
| Bascule avec scène / tableau blanc | Ne doit pas casser définitivement ces modes |
