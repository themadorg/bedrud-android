Après connexion, le tableau de bord est le point central pour créer des salles, les rejoindre et retrouver celles utilisées récemment.

## Créer une salle

1. Depuis le tableau de bord, créez une nouvelle salle (nom et options exposées par votre hôte).
2. La salle apparaît dans votre liste.
3. Ouvrez la salle pour entrer dans le flux pré-join / réunion.

Les cartes de salle proposent rejoindre et supprimer. La densité de mise en page peut varier selon la version de l'UI, mais rejoindre et supprimer restent disponibles.

## Rejoindre rapidement

Collez un **slug de salle** ou une **URL de réunion complète** dans le champ de rejoindre rapide. Bedrud résout la salle et vous mène au bon chemin d'accès.

Les liens partageables ressemblent en général à :

```
https://your-host/m/<room-id-or-slug>
```

## Salles récentes

Après avoir **rejoint** une salle, elle est ajoutée aux **Salles récentes**.

- La liste **survit à un rechargement complet** de la page.
- **Retirez** une entrée récente — elle reste absente après rechargement.
- Utilisez une carte récente pour rouvrir cette salle rapidement.

Les salles récentes sont une liste de confort côté client ; elles ne remplacent pas la liste de salles côté serveur que vous possédez ou auxquelles vous avez accès.

## Page d'accueil

Lorsque vous ouvrez la racine de l'instance :

- **Déconnecté** — chemin marketing / connexion (pas de proxy API cassé)
- **Connecté** — entrée du tableau de bord
