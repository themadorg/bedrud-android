أنشئ حسابًا، واحتفظ بجلستك، واضبط الملف الشخصي وتفضيلات الوسائط قبل الانضمام إلى مكالمة.

## التسجيل وتسجيل الدخول

1. افتح نسخة Bedrud في المتصفح.
2. **سجّل** بالبريد وكلمة المرور (وأي رمز دعوة يطلبه المضيف).
3. إذا كان التحقق من البريد مفعّلًا، أكمل خطوة التحقق عندما تطلبها الواجهة.
4. **سجّل الدخول** ببيانات صحيحة. كلمة المرور الخاطئة تظهر خطأ واضحًا ولا تنشئ جلسة.
5. **سجّل الخروج** من قائمة الحساب. المسارات المحمية (لوحة التحكم، الإعدادات، المشرف) تعيد التوجيه للمصادقة.

تبقى الجلسات بعد **إعادة تحميل** الصفحة كاملة طالما رمز الوصول صالحًا.

### خيارات دخول أخرى

حسب إعداد النسخة، قد ترى أيضًا:

- **OAuth** (Google أو GitHub أو Twitter أو OIDC مخصص)
- **مفاتيح المرور (Passkeys)** (WebAuthn) بعد تسجيلها في إعدادات الأمان
- **انضمام الضيف** للغرف التي تسمح بذلك — بدون حساب (انظر [الانضمام إلى اجتماع](/ar/docs/using/joining-a-meeting))

نسيت كلمة المرور؟ استخدم **نسيت كلمة المرور** في صفحة الدخول ثم اتبع رابط إعادة التعيين (يجب ضبط SMTP على الخادم).

لتفاصيل المشغّل (JWT، مفاتيح المرور، حدود المعدل) انظر [المصادقة](/ar/docs/backend/authentication).

## الملف الشخصي

افتح **الإعدادات → الملف الشخصي** من لوحة التحكم.

- عدّل **الاسم** / حقول العرض واحفظ — تُحمَّل القيم بشكل صحيح بعد إعادة التحميل.
- تظهر التغييرات في لوحة التحكم وبلاطات الاجتماع وقوائم المشاركين.

## الصورة الرمزية

ما زالت تحت **الإعدادات → الملف الشخصي**:

| القاعدة | التفاصيل |
|---------|----------|
| الصيغ | JPEG، PNG، WebP |
| حد الحجم | أقل من **٢ ميجابايت** |
| الحجم الزائد / MIME غير صالح | يُرفض بخطأ (بدون تعطّل) |
| إعادة الرفع | يستبدل الملف السابق |

بعد الرفع يتحدّث المعاينة فورًا. أكّد الصورة الجديدة بعد إعادة التحميل في لوحة التحكم والاجتماع.

## لوحات الإعدادات

افتح كل لوحة من مسار إعدادات لوحة التحكم:

| اللوحة | ماذا تفعل |
|--------|-----------|
| **الملف الشخصي** | الاسم وهوية العرض |
| **الصوت** | قوائم الميكروفون / السماعة؛ تفعيل **اضغط للتحدث** + التقاط مفتاح الضغط؛ يُحفظ عبر إعادة التحميل |
| **الفيديو** | اختيار الكاميرا؛ يستمر عبر إعادة التحميل |
| **الأمان** | تغيير كلمة المرور وتسجيل مفاتيح المرور |
| **المظهر** | السمة ومفاتيح الواجهة ذات الصلة (عند ظهورها) |
| **تجريبي** | ميزات اختيارية مثل **السبورة المشتركة** و**حفل مشاهدة YouTube** |

تُخزَّن التفضيلات لحسابك (وقد تُزامن إلى غلاف الاجتماع عند الانضمام). انظر [الصوت واضغط للتحدث](/ar/docs/using/audio-and-push-to-talk) لسلوك PTT أثناء المكالمة.
