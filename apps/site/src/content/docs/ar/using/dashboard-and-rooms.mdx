بعد تسجيل الدخول، لوحة التحكم هي نقطة الانطلاق لإنشاء الغرف والانضمام إليها والعودة إلى الغرف التي استخدمتها مؤخرًا.

## إنشاء غرفة

1. من لوحة التحكم أنشئ غرفة جديدة (الاسم وأي خيارات غرفة يعرضها المضيف).
2. تظهر الغرفة في قائمتك.
3. افتح الغرفة للدخول إلى تدفق ما قبل الانضمام / الاجتماع.

تدعم بطاقات الغرف إجراءات الانضمام والحذف. قد تختلف كثافة التخطيط حسب إصدار الواجهة، لكن الانضمام والحذف يبقيان متاحين.

## الانضمام السريع

الصق **slug الغرفة** أو **رابط الاجتماع الكامل** في الانضمام السريع. يحلّ Bedrud الغرفة ويأخذك إلى مسار الانضمام الصحيح.

تبدو الروابط القابلة للمشاركة عادةً هكذا:

```
https://your-host/m/<room-id-or-slug>
```

## الغرف الأخيرة

بعد **الانضمام** إلى غرفة تُضاف إلى **الغرف الأخيرة**.

- القائمة **تبقى بعد إعادة تحميل الصفحة كاملة**.
- **أزل** إدخالًا أخيرًا — يبقى محذوفًا بعد إعادة التحميل.
- استخدم بطاقة أخيرة لإعادة فتح تلك الغرفة بسرعة.

الغرف الأخيرة قائمة مريحة على العميل؛ لا تحل محل قائمة الغرف من جانب الخادم التي تملكها أو يمكنك الوصول إليها.

## الصفحة الرئيسية

عند فتح جذر النسخة:

- **غير مسجّل** — مسار تسويقي / تسجيل دخول (بدون وكيل API معطوب)
- **مسجّل** — دخول إلى لوحة التحكم
