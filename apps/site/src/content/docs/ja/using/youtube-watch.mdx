YouTube ウォッチでは、1 人の参加者が動画を共有し、ルーム内の全員が一緒に視聴できます。

## 機能を有効にする

YouTube ウォッチパーティーは **設定 → 実験的 → YouTube ウォッチパーティー** の後ろにゲートされる場合があります。共有コントロールが現れることを期待する前に有効にします。

## 動画を共有

1. 会議コントロールから**共有** / YouTube ダイアログを開きます。
2. **有効な YouTube URL** を貼り付けます。
3. 視聴**オーバーレイ**があなたと他の参加者で再生されるはずです。

## 再生同期

再生、一時停止、シーク（実装どおり）はリモート視聴者で**おおよそ同期**したままであるべきです。低品質ネットワークではフレーム単位のロックは保証されません；小さなずれが出ることがあります。

## エラーと終了

| 状況 | 期待 |
|------|------|
| 無効な URL | 明確なエラー；空白のハングなし |
| ウォッチを閉じる | 双方でオーバーレイが消える |
| ステージ / ホワイトボードとの切り替え | それらのモードを恒久的に壊さない |
