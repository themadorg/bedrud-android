После входа дашборд — база для создания комнат, входа в них и возврата к недавно использованным.

## Создание комнаты

1. С дашборда создайте новую комнату (имя и любые опции, которые показывает хост).
2. Комната появится в списке.
3. Откройте комнату, чтобы перейти к pre-join / встрече.

Карточки комнат поддерживают вход и удаление. Плотность раскладки может отличаться по версии UI, но вход и удаление остаются доступны.

## Быстрый вход

Вставьте **slug комнаты** или **полный URL встречи** в быстрый вход. Bedrud находит комнату и ведёт вас по нужному пути.

Ссылки для шаринга обычно выглядят так:

```
https://your-host/m/<room-id-or-slug>
```

## Недавние комнаты

После **входа** в комнату она добавляется в **Недавние комнаты**.

- Список **переживает полную перезагрузку** страницы.
- **Удалите** недавнюю запись — она останется удалённой после перезагрузки.
- Карточка недавней комнаты быстро открывает её снова.

Недавние комнаты — удобный клиентский список; он не заменяет серверный список комнат, которыми вы владеете или к которым имеете доступ.

## Лендинг

При открытии корня экземпляра:

- **Вышли** — маркетинговый путь / вход (без сломанного API-прокси)
- **Вошли** — вход на дашборд
