YouTube Watch позволяет одному участнику поделиться видео, чтобы все в комнате смотрели вместе.

## Включение функции

YouTube watch party может быть за **Настройки → Экспериментальное → YouTube watch party**. Включите до того, как ожидаете появление контрола шаринга.

## Поделиться видео

1. Откройте диалог **share** / YouTube из контролов встречи.
2. Вставьте **валидный URL YouTube**.
3. Оверлей **watch** должен воспроизводиться у вас и у других участников в комнате.

## Синхронизация воспроизведения

Play, pause и seek (как реализовано) должны **примерно оставаться в синхроне** для удалённых зрителей. Покадровый lock не гарантируется на плохих сетях; возможен небольшой дрейф.

## Ошибки и закрытие

| Ситуация | Ожидание |
|----------|----------|
| Невалидный URL | Понятная ошибка; без пустого зависания |
| Закрыть watch | Оверлей исчезает у обеих сторон |
| Переключение со stage / whiteboard | Не должно навсегда ломать эти режимы |
