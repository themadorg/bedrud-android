Настройте микрофоны и динамики, затем при желании используйте push-to-talk (PTT), чтобы передавать только при удержании клавиши.

## Выбор устройств

В **Настройки → Аудио** (и на pre-join):

- Списки микрофона и динамика заполняются, когда браузер даёт разрешения.
- Смена устройств **mid-call** должна работать без полного reconnect комнаты.

Камеры — в **Настройки → Видео**. См. [Аккаунт и настройки](/ru/docs/using/account-and-settings).

## Включение push-to-talk

1. Откройте **Настройки → Аудио**.
2. Включите **push-to-talk**.
3. Захватите **hold-клавишу** контролом key capture.
4. Сохраните, если UI требует явного save — предпочтения должны переживать перезагрузку.

## На встрече

| Шаг | Ожидание |
|-----|----------|
| Удерживать PTT-клавишу | Микрофон **передаёт**; другие вас слышат |
| Отпустить клавишу | Микрофон **muted**; другие перестают слышать |
| Индикаторы | Локальные контролы и удалённые плитки могут показывать состояние PTT |
| Отключить PTT | Возвращается обычный mute-toggle |

Не оставляйте PTT включённым с застрявшим открытым микрофоном, когда хотите молчать — отключите PTT или mute явно.

## Deafen / наушники

Если сборка даёт **deafen** или headphone-контролы, они не должны оставлять удалённое аудио застрявшим для других. Deafen — локальное состояние прослушивания, не удалённый mute других людей (если не сочетается с модерацией).
