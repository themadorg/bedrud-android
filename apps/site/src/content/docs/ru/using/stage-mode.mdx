Режим сцены меняет раскладку комнаты: ведущий (или набор сцены) на переднем плане вместо обычной сетки или spotlight.

## Запуск сцены

1. Как хост (или роль с правом управления сценой), **запустите сцену** из контролов встречи.
2. Остальные участники видят **раскладку сцены** и любой **join-notifier**, предназначенный для stage-сессий.
3. Полоса / боковая панель участников может по-прежнему показывать остальных (переключите, если есть).

## Демонстрация экрана на сцене

- Поделитесь экраном при активной сцене — другие видят **stage screen-share overlay**.
- **Остановка шаринга** снимает оверлей.

Демонстрация экрана вне сцены (обычная сетка): [Управление встречей](/ru/docs/using/meeting-controls).

## Завершение сцены

**End stage** возвращает комнату к обычной сетке или spotlight-раскладке.

## Поздние входы

Тот, кто входит **пока сцена уже активна**, должен сразу видеть корректное состояние сцены — не пустую сетку навсегда. Если поздний вход видит неверную раскладку, перезагрузите один раз; если снова сбой — сообщите как регрессию.

## Советы

- Сцена рассчитана на демо, обучение и one-to-many презентации.
- Комбинируйте с [YouTube Watch](/ru/docs/using/youtube-watch) или [Whiteboard](/ru/docs/using/whiteboard) только когда оверлеи рассчитаны на совместную работу; переключайте осторожно, чтобы участники не застряли под несколькими полноэкранными слоями.
