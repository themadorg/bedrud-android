Tras iniciar sesión, el panel es la base para crear salas, unirte a ellas y volver a las que usaste recientemente.

## Crear una sala

1. Desde el panel, crea una sala nueva (nombre y las opciones de sala que exponga tu anfitrión).
2. La sala aparece en tu lista.
3. Ábrela para entrar en el flujo de pre-unión / reunión.

Las tarjetas de sala admiten unirse y eliminar. La densidad del diseño puede variar según la versión de la UI, pero unirse y eliminar siguen disponibles.

## Unión rápida

Pega un **slug de sala** o una **URL de reunión completa** en la unión rápida. Bedrud resuelve la sala y te lleva al camino de unión correcto.

Los enlaces compartibles suelen verse así:

```
https://your-host/m/<room-id-or-slug>
```

## Salas recientes

Después de **unirte** a una sala, se añade a **Salas recientes**.

- La lista **sobrevive a una recarga completa** de la página.
- **Elimina** una entrada reciente — permanece ausente tras recargar.
- Usa una tarjeta reciente para reabrir esa sala con rapidez.

Las salas recientes son una lista de comodidad en el cliente; no sustituyen la lista de salas del servidor que posees o a las que tienes acceso.

## Página de inicio

Al abrir la raíz de la instancia:

- **Sin sesión** — camino de marketing / acceso (sin proxy de API roto)
- **Con sesión** — entrada al panel
